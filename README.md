# Sabina's Birthday Adventure 🎉

Welcome to **Sabina's Birthday Adventure**, an interactive 2D RPG-style Android game created to celebrate Sabina's special day!

## 🌟 Introduction
In this game, Sabina embarks on a mini-adventure roaming a beautiful, vibrant village on her birthday. She will encounter friendly villagers, mythical creatures, animals, and magical gifts waiting to be discovered. 

### Features:
*   **Interactive Village Map:** Roam freely around a map populated with houses, farm animals, and beautiful scenery.
*   **Custom Movement:** Control Sabina using the on-screen joystick (or WASD/Arrow keys when using a physical keyboard).
*   **Animated Character:** Sabina features basic frame-based sprite animations indicating when she is idling, walking, and interacting.
*   **Proximity Dialogues:** Walk up to NPCs, animals, or items to reveal special birthday wishes!
*   **Adventure Dialogue Log:** Access the story log to review the kind words Sabina receives from everyone along the adventure.

## 📱 Installation Manual

Since this app was built in the Google AI Studio Build environment, you can download the APK directly to install it on your Android phone.

### Step 1: Download the APK
1. In the AI Studio Build interface, look for the settings menu (typically the three dots or gear icon in the corner).
2. Select the option to **Generate APK** or **Download App**.
3. Save the generated `.apk` file directly to your Android device, or to your computer first.

### Step 2: Transfer to your Android Device (If downloaded on PC)
* Connect your Android phone to your computer via a USB cable.
* Copy the APK file into your phone’s `Downloads` folder.

### Step 3: Install the App
1. On your Android device, open an Explorer or File Manager application and locate the downloaded `.apk` file.
2. Tap the APK file to begin the installation.
3. **Important Note:** If this is your first time installing an app outside the Google Play Store, your phone will prompt you to enable "Install unknown apps" or "Allow installation from unknown sources". 
   * Tap on **Settings** in the prompt.
   * Toggle the switch for your File Manager/Browser to **Allow** or turn it on.
   * Go back and tap **Install** again.
4. Once the installation finishes, tap **Open** to start playing Sabina's birthday adventure!

Enjoy the celebration! 🎈
